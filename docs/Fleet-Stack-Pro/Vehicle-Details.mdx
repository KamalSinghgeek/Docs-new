The **Vehicle Details** feature allows you to access specific information about each vehicle in your fleet.

### **How to Access Vehicle Details**:

- **Tap a Vehicle**: Select any vehicle on the map to open detailed information.
  - **Info**: View basic details like vehicle type, model, and registration information.
  - **History**: Check the historical data of the vehicle, including past routes and performance.
  - **Sensors**: Monitor sensor data, such as engine temperature, tire pressure, and fuel levels.
  - **Details**: Get in-depth information regarding the current status of the vehicle, including location, speed, and condition.

### **Why Vehicle Details Matter**:

- **Efficient Fleet Management**: Quickly access essential data for decision-making.
- **Monitor Performance**: Keep track of historical data and real-time sensor readings.
- **Maintain Vehicle Health**: Identify maintenance needs by tracking sensor data and vehicle status.
