The **Support and Feedback** feature allows you to get assistance for any issues you're facing with the Fleet Stack Pro app, as well as provide valuable feedback to improve the service.

### **How to Access Support and Feedback**:

- Open the **Other Navigation Menu** within the Fleet Stack Pro app.
- Select the **Support and Feedback** option to access the help and feedback section.

Using the **Support and Feedback** feature ensures that you can easily resolve any issues and contribute to the continuous improvement of the app.
