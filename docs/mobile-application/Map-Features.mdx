The **Map Features** section allows users to view the real-time location of all vehicles, interact with vehicle icons for detailed information, and access travel history for each vehicle.

---

### **Accessing the Map Page**:

- **View all vehicles clustered by location**: The map displays vehicles grouped by their geographical location, providing an overview of your fleet’s positions.
- **Select map type**: You can choose from various map types for different views:
  - Default
  - Hybrid
  - Satellite
  - Terrain
- **Zoom in and out**: Adjust the zoom level for a more detailed or broader view of the vehicles on the map.

### **Interacting with Vehicle Icons**:

- **Tap a vehicle icon**: When you tap on a vehicle icon, an **Info Window** opens, providing the following details:
  - Vehicle Number
  - Status (Running, Stopped, Idle)
  - Ignition Status
  - Today’s Distance
  - Speed
  - Address

### **Vehicle Travel History**:

- **View travel history**: Tap on the **Date** to access the detailed travel information for that day:
  - Start Point
  - Running Time and Distance
  - Stoppages
