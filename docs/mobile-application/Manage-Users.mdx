The **Manage Users** section allows administrators to manage user access and details within the system. It is essential for maintaining control over who can view and manage vehicle data.

---

### **View User List**:

- **View all users**: The system displays a list of users along with their names and profile details.
- **Edit user details**: You can click on a user’s profile to update their information, such as names or roles, as necessary.

### **Add New Users**:

- **Add New User**: You can add new users to the system by selecting the **Add New User** button. This allows you to enter the necessary user details and assign them the appropriate permissions.

This section ensures that administrators can easily add, view, and manage users with the necessary permissions to operate the system effectively.

This functionality enables administrators to control user access, ensuring that the right people have access to sensitive fleet data and features.
